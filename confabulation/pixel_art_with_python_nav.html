简介

这是个索引页面，您可以在上面的目录检索所有《鳞性相的像素画创作辅助》系列博文。

本系列是个人在像素画创作中使用 Python 辅助创作的经验分享。所谓辅助创作，最常见的包括替色、提取调色板、检查风格的一致性及视觉效果、生成展示图片等。这些工作一般相当重复、有时繁琐，能交由脚本完成自然最好。如果您也和我一样只是出于兴趣创作像素画（尤其是作为游戏资源的小画布像素画）、且可用于此的时间并不多的话，大抵也能体会被一些重复工作无谓占用时间的苦恼。

当然，如果您可供创作的时间很多，也不妨碍阅读这些博文；毕竟少做任何一项重复工作都是预防早发性痴呆的好实践[来源请求]。不过，本系列博文假定读者已有一定的语言基础——如果您没有，可以先去学习一些基础的语法，以便能够看懂简单的 Python 代码：这一般不需要太长时间。

本系列博文会经常使用 NumPy 和 Pillow 两个第三方库。所以请默认在所有代码片段中，他们已经导入过了：

import numpy as np
from PIL import Image as im

我会对函数的参数及返回值进行类型标注，因此会暴力导入标准库 typing 以及一些其他东西。这些并不是必要的：

from numbers import Real
from typing import *

最后，撰写本系列博文时，使用的 Python 版本为 3.8.8，但相关代码一般能向高版本兼容。
